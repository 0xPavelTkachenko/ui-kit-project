# UI Kit
Набор элементов для создания сайта.

## Клонирование репозитория
```console
$ git clone https://github.com/p-tka4enko/ui-kit-project.git
```

## Сборка проекта
```console
$ cd ui-kit-project
$ npm i
$ npm run build
```

## Запуск линтера
```console
$ npm run lint
```

## Запуск сервера
```console
$ npm run server
```
