# UI Kit
Набор элементов для создания сайта.

## Сборка проекта
```console
$ cd ui-kit-project
$ npm i
$ npm run build
```

## Запуск линтера
```console
$ npm run lint
```

## Запуск сервера
```console
$ npm run server
```
